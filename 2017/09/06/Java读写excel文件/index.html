Java读写excel文件

Sep 6, 2017

最近做了一个退款的需求，根据XXX提供的userid，orderid，refundmoney等数据调用相应的接口退款，数据给的形式不是txt，而是excel表格的形式。之前没做过，在此记录下。

选用的库是Apache POI，提供对excel的读写功能的java api。

  • HSSF：excel2003（.xls）
  • XSSF：excel2007及以后（.xlsx）

对应的maven依赖是

XLSReader

XLSWriter
