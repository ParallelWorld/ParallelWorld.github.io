Java读写excel文件

最近做了一个退款的需求，根据 XXX 提供的 userid，orderid，refundmoney 等数据调用相应的接口退款，数据给的形式不是 txt，而是 excel 表格的形式。之前没做过，在此记录下。

选用的库是 Apache POI，提供对 excel 的读写功能的 java api。

  • HSSF：excel2003（.xls）
  • XSSF：excel2007 及以后（.xlsx）

对应的 maven 依赖是

XLSReader

XLSWriter
